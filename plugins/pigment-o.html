Pigment.O is a Colour Picker and Colour Mixer. An agnotic colour calculator that uses the document as a reference. Constructed to be a universal soluton inside docker.
Modular UI for each workflow and fully scalable. Makes use of presets to simplify settings.
Harmony rules to create colour sets dynamically within a preferred space.
Panels with several graphical interfaces. Utility ranges from complementary, classical, and even original.
Channel sliders allow fine adjustment within their own respective spaces.
Mixer calculates the interpolation of custom colours within their respective space.
Pinned colours are a small temporary palette of saved colours that can be recalled easily.
History keeps track of all selected colours to ensure return to a previous selection.
Hex code of the colour regardless of space or range. Quickly share hex codes with fast copy-and-paste. Recognizes many colour formatting types and also colours with name.
Settings for slower performinc systems.
API allows direct interaction from other plugins.
